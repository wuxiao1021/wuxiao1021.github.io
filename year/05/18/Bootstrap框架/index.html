布局容器

Bootstrap 需要为页面内容和栅格系统包裹一个 .container 容器。我们提供了两个作此用处的类。注意，由于 padding 等属性的原因，这两种 容器类不能互相嵌套。

.container 类用于固定宽度并支持响应式布局的容器
.container-fluid 类用于 100% 宽度，占据全部视口（viewport）的容器。流式布局容器

栅格系统

Bootstrap 提供了一套响应式、移动设备优先的流式栅格系统，随着屏幕或视口（viewport）尺寸的增加，系统会自动分为最多12列。它包含了易于使用的预定义类，还有强大的mixin 用于生成更具语义的布局。

自动 col-xs-* 百分百布局必须.container在容器中；

Bootstrap 标签；

Bootstrap类名；给标签添加类名改变样式

h1 -h6:标题类名；

lead:突出某个段落；

pre-scrollable只需要在pre标签或多个标签上添加类名“.pre-scrollable”，就可以控制代码块区域最大高度为340px，一旦超出这个高度，就会在Y轴出现滚动条。

列表类名

ist-unstyled：清除ul ol 默认样式；

颜色类

text-muted：提示，使用浅灰色（#999）

text-primary：主要，使用蓝色（#428bca）

text-success：成功，使用浅绿色(#3c763d)

text-info：通知信息，使用浅蓝色（#31708f）

text-warning：警告，使用黄色（#8a6d3b）

text-danger：危险，使用褐色（#a94442）

文本对齐类

text-left：左对齐

text-center：居中对齐

text-right：右对齐

text-justify：两端对齐

bootstrap 图片类：

img-responsive:响应式图片，主要针对于响应式设计；

img-rounded:给图片一个6px的倒角；

img-circle:图片倒圆；

ing-thumbnail:缩略图片，给图片加个类似相框的边框；
