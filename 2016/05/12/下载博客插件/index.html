下载插件创建博客

一、下载Noed.js

1. Noed.js下载地址 点这下载

a. 格式下载：v4.4.4LTS是稳定版本；（工作中使用）

b. 格式下载：v6.1.0CUrrent(经常更新不建议在工作中使用)

2. 下载安装完成之后，打开终端$ node -v 显示版本号 即安装成功

一、下载hexo博客模块

1. hexo博客模块是一个第三方插件，能生成一个静态博客网站框架

2. hexo模块下载地址 //www.npmjs.com打开网址之后在搜索框搜索hexo下载；

3. 下载完成之后在终端 $ sudo npm install -g hexo:安装hexo模块

二、开启终端软件创建自己的博客（前提：熟练终端命令操作）

1. $ ls:显示当前目录

2. $ cd:进入桌面文件夹（ Desktop ）

3. $ mkdir:创建文件夹（ blog ）输入cd:进入（ blog ）文件夹

4. $ hexo init:初始化博客目录

5. $ hexo server:重新启动终端服务之后复制地址 http://localhost:4000/打开就能在本地看到网站效果啦！之后我们需要创建自己的第一篇博客

6. $ hexo new 你要创建的博客名称:这样就成功创建啦自己的.md格式第一篇博客啦！是不是很激动呀，那么我们怎么找到他呢？打开博客文件夹blog/source/-posts/这里就能看到你自己创建的博客.md文件

三、申请github二级域名，上传到线上

申请二级域名

1. 打开终端执行 $ git config –global user.name “Your Name” 创建设置用户名

2. $ git config –global user.email “email@example.com” 创建设置邮箱

3. 执行玩以上两个命令会出现一个输入密码：设置自己的密码。注意输入密码页面是不会提示的；

4. 进入 github.com登入自己账号密码

5. 在主页右上角有个加号标签按钮点击里面的 New reposltory 创建仓库；注意：仓库名的写法 用户名.github.io 如：（ zhangsan.github.io ）一定要这么取名；

绑定 SSH

1. 点击主页右上角加号旁边下拉按钮； Setting 选择左边 SSH and GPG keys>New SSH key这里面有两个需要填写的信息title自己起个名字，用来辨别是哪台机器

2.key获取 输入克隆命令 ssh-keygen ，输入 vim ~/.ssh/id_rsa.pub
获取Key复制生成的key 如下：复制完之后一定要退出 key 命令 （ :q ）

3.$ git clone 加仓库地址:下载线上仓库地址

4.$ git status :查看当前的创客状态

5.$ git add 加 .代表所有博客文件；或 加博客文件名 :选择博客所要上传的文件；

6. $ git commit :提交到暂存区

7. $ git remote add origin :提交到暂存区

7. $ git remote add origin :远程链接仓库

8. $ git push -u origin master :把本地博客分支移动到远程分支

5. 输入账号：$ git config –global user.name “用户名”

6. 输入邮箱：$ git config –global user.email “邮箱”

修改 _config.yml 上传地址

1.打开博客目录 _config.yml 找到 deploy：参数；如下：保存；

马上就能上线看到自己的博客啦，好激动呀!

发布

1. 在终端找到博客文件夹，在博客文件夹下面执行 $ hexo g ：把本地博客文件转换成html文件等；

2. 在终端找到博客文件夹，在博客文件夹下面执行 $ hexo d ：把转换之后的博客文件推送到自己的github仓库，并发布。
